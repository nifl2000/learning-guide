# Software Engineering Lernpfad

Ein umfassender Lernpfad mit 182 verlinkten Diagrammen zu Software Engineering Themen.

## 🌐 Live Website

Verfügbar unter: https://nifl2000.github.io/learning-guide/

## 🛠️ Technologie

- **MkDocs** mit Material Theme
- Automatisches Deployment via GitHub Actions

## 📁 Struktur

```
learning-guide/
├── mkdocs.yml              # MkDocs Konfiguration
├── requirements.txt        # Python Dependencies
├── docs/
│   ├── index.md           # Startseite
│   ├── images/            # 201 PNG-Diagramme
│   └── *.md               # Kategorien
└── .github/workflows/     # CI/CD
```

## 📊 Inhalte

- **System Design** - Architektur, Microservices, Caching
- **Databases** - SQL, NoSQL, Redis, Kafka
- **Cloud & DevOps** - Docker, Kubernetes, CI/CD
- **Networking** - HTTP, TCP/IP, REST, gRPC
- **Security** - OAuth, JWT, Encryption
- **Operating Systems** - Linux, Memory, File Systems
- **Programming** - Git, Design Patterns, Testing
- **Career** - Interviews, Roadmaps
- **AI** - ChatGPT, Machine Learning

## 📊 Statistik

- **182 Themen** mit Diagrammen
- **10 Kategorien**
- **3 Prioritätsstufen**: MUST KNOW, SHOULD KNOW, NICE TO KNOW

## 🚀 Lokale Entwicklung

```bash
pip install -r requirements.txt
mkdocs serve
```
